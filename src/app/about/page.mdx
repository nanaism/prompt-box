# Prompt Box について
 
> AI プロンプトテンプレートを管理・カスタマイズするためのツール
 
## Prompt Box とは
 
Prompt Box は、AI プロンプトテンプレートの作成、管理、カスタマイズを支援するパワフルなアプリケーションです。
 
コンテンツクリエイター、開発者、AI マニアを問わず、
Prompt Box を使えば簡単に最適なプロンプトを作成できます。
 
**プロンプト作成をより効率的に、そして楽しく管理することを目指しています** 👍
 
<Card className="px-4 bg-gradient-to-br from-slate-50 to-purple-50 dark:from-zinc-900 dark:to-purple-950">
 
## 💡 特徴
 
- **テンプレートライブラリ**：目的別に整理された、プロンプトテンプレートコレクション
- **変数カスタマイズ**：フォーム入力だけで簡単にテンプレートをカスタマイズ
- **リアルタイムプレビュー**：変更に応じたプロンプトの更新をすぐに確認
- **ワンクリックコピー**：生成したプロンプトをすぐに使用できるようにコピー機能を搭載
- **自動保存と評価**：使用したプロンプトを自動保存し、後から結果の評価を記録
